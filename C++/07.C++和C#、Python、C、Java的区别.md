# 07.C++和C#、Python、C、Java的区别

+ python

  + python脚本语言，可跨平台 --- c++编译语言，需要特定平台，但是快
  + python缩进 --- c++花括号
  + python变量不需要事先定义类型 --- c++定义变量要事先定义类型

+ C

  + C语言不能重载 --- C++支持同名函数，支持重载

+ Java

  + 语言特性：Java无指针、安全、并且由于JVM可移植性强、Java有接口取代了C++的抽象类

  + 垃圾回收：自动回收 --- 析构函数

+ C#

  + 语言特性：无指针、有接口、安全
